# Ảnh local dự phòng

Khi chưa dùng Supabase, đặt ảnh theo cấu trúc:

- `img/C1-ELAN/0.jpg`, `1.jpg`...
- `img/C1-NOIR/0.jpg`...
- `img/C8-THE-ART/0.jpg`...
- `img/C9-VELVET/0.jpg`...
- `img/C10-MIDNIGHT/0.jpg`...
- `img/C12-AMOR/0.jpg`...
- `img/C12-ROMA/0.jpg`...

Ảnh số `0.jpg` là ảnh đại diện. Khi Supabase đã có `room_images`, website ưu tiên ảnh live và không cần bộ ảnh local này.
